# repodesc
Python script to print the description of a Github repository

## Usage

```
    $ repodesc rolandcrosby/repodesc
    get the description of a github repo
    
    $ repodesc rolandcrosby/repodesc.git
    get the description of a github repo
    
    $ echo "rolandcrosby/repodesc" | repodesc
    get the description of a github repo
```
